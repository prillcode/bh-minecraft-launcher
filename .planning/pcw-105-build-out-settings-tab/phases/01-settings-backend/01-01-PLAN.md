<task type="auto">
  <name>Task 2: Add new IPC handlers to ipc-handlers.ts</name>
  <files>src/main/ipc-handlers.ts</files>
  <action>
Add the following imports at the top of the file:
- `import { getLauncherPaths } from '../core/utils/paths';`
- `import type { LauncherSettings } from '../core/settings';` (needed for key cast in settings:set)
- `import { shell, app } from 'electron';` — extend the existing `import { BrowserWindow, type IpcMain } from 'electron'` to also include `shell` and `app`

In the Settings section (after the existing handlers), add four new handlers:

**settings:set** — generic key/value setter:
```
ipcMain.handle('settings:set', async (_event, key: keyof LauncherSettings, value: unknown) => {
  setSetting(key, value as LauncherSettings[typeof key]);
  return { success: true };
});
```

**settings:get-app-info** — returns version + data path (used by About section):
```
ipcMain.handle('settings:get-app-info', async () => {
  const paths = getLauncherPaths();
  return {
    version: app.getVersion(),
    dataPath: paths.root,
  };
});
```

**settings:open-data-folder** — opens userData in OS file manager:
```
ipcMain.handle('settings:open-data-folder', async () => {
  const paths = getLauncherPaths();
  await shell.openPath(paths.root);
  return { success: true };
});
```

**settings:clear-cache** — removes temp/ and natives/ directories:
```
ipcMain.handle('settings:clear-cache', async () => {
  const paths = getLauncherPaths();
  await fs.rm(paths.temp, { recursive: true, force: true });
  await fs.rm(paths.natives, { recursive: true, force: true });
  logger.info('Cache cleared (temp + natives)');
  return { success: true };
});
```

Keep the existing `settings:set-default-auth-mode` handler — `auth-store.ts` in the renderer calls it directly.
  </action>
