Replace the stub file with a full implementation. Read the stub first.

Props: `{ onClose: () => void; onAdd: (server: ServerInfo) => void }`

State: `name: string`, `host: string`, `port: string` (default `'25565'`), `notes: string`, `saving: boolean`, `error: string | null`.

Form fields:
- **Server Name** (required) — text input, id="as-name", autoFocus
- **Host** (required) — text input, id="as-host", placeholder "mc.example.com"
- **Port** — number input, id="as-port", min=1, max=65535, default value "25565"
- **Notes** (optional) — textarea, id="as-notes", rows=2, placeholder "Optional notes..."

Modal structure: use the exact same CSS classes as CreateInstanceModal:
- `.modal-overlay` wrapping div (overlay ref for click-to-close)
- `.modal` inner div with `role="dialog"` `aria-modal="true"`
- `.modal__header` with `.modal__title` "Add Server" and `.modal__close` button
- `.form-group` for each field
- `.modal__actions` with Cancel (btn--ghost) and Add Server (btn--primary, disabled while saving)
- Escape key listener to call onClose

Error display: `{error && <p style={{ color: 'var(--danger)', fontSize: '12px', marginBottom: '12px' }}>{error}</p>}`
