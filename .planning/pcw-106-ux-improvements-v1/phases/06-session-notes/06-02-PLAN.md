# 06-02 — NotesManager Backend

## Objective
Build the persistence and IPC layer for Session Notes. Mirrors the ModManager pattern
exactly: an electron-store-backed class, a types file, IPC handlers registered in
ipc-handlers.ts, and a preload bridge. The `notes:list-screenshots` channel reuses
the screenshot-listing logic from the removed `worlds:list-screenshots` handler.

## Context
- @src/core/mods/mod-manager.ts — pattern to mirror (electron-store, Store generic, key paths)
- @src/core/mods/types.ts — example types file pattern
- @src/main/ipc-handlers.ts — where to register IPC handlers (append near mod handlers)
- @src/main/preload.ts — where to add `notes` to contextBridge
- @src/renderer/global.d.ts — where to declare `launcher.notes` and `NoteEntry` / `ScreenshotInfo`
- @src/core/game/instance.ts — `getInstance(id)` helper to resolve `gameDirectory` for screenshots

## Data model
```typescript
// src/core/notes/types.ts
export interface NoteEntry {
  id: string;           // uuid v4
  instanceId: string;
  title: string;
  text: string;
  screenshotPaths: string[];  // absolute paths on disk
  createdAt: number;    // Date.now()
  updatedAt: number;
}
```

Store layout (electron-store name: `"notes"`):
```
notes[instanceId][entryId] = NoteEntry
```
Key path pattern: `notes.${instanceId}.${entryId}` — identical to ModManager's `mods.${instanceId}.${projectId}`.

### Task 1 — Create NotesManager class
- type: create
- file: src/core/notes/notes-manager.ts
- action: Create the class mirroring ModManager. Methods:
  - `list(instanceId): Promise<NoteEntry[]>` — return all entries for instance, sorted by `updatedAt` desc
  - `get(instanceId, entryId): Promise<NoteEntry | undefined>`
  - `create(instanceId, entry: Omit<NoteEntry, 'id' | 'createdAt' | 'updatedAt'>): Promise<NoteEntry>` — generate id with `crypto.randomUUID()`, set timestamps

  - `delete(instanceId, entryId): Promise<void>`
- Store instantiation: `new Store<{ notes: Record<string, Record<string, NoteEntry>> }>({ name: 'notes', defaults: { notes: {} } })`
- Use `crypto.randomUUID()` (Node built-in, no extra package needed)
