# 06-01 ‚Äî Remove My Worlds

## Objective
Scrub all My Worlds code from the codebase cleanly: nav item, route, component,
IPC handlers, preload bridge, types, and CSS. Phase 03 shipped this feature but
it is useless for a server-only community launcher ‚Äî no singleplayer saves ever exist.
This plan removes all traces so phase 06-02 can build Session Notes in their place.
