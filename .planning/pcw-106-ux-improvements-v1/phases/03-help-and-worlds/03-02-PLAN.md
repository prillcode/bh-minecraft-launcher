# Plan: 03-02 — My Worlds IPC Backend

**Phase:** 03 — New Bottom Panel Group: Help + My Worlds
**Work:** pcw-106 ux-improvements-v1

## Context

My Worlds shows the user's Minecraft save folders for the selected instance, with notes
and screenshots. All data is local — no network calls.

**Data sources:**
- **World list:** `{instance.gameDirectory}/saves/` — one subfolder per world
- **World name:** folder name (skip NBT parsing for now — use folder name as display name)
- **Last played:** folder `mtime` from `fs.stat()`
- **User notes:** electron-store keyed by `{instanceId}/{worldFolder}`
- **Screenshots:** `{instance.gameDirectory}/screenshots/*.png` files

**Screenshot display strategy:**
Register a custom `launcher-file://` Electron protocol in `main/index.ts`.
The renderer loads screenshots as `<img src="launcher-file:///absolute/path/to/file.png">`.
This is secure (read-only, limited to specified paths) and avoids base64 encoding overhead.

## Steps

### Step 1 — Register `launcher-file://` protocol in `main/index.ts`

**File:** `src/main/index.ts`

Add import at top:
```ts
import { protocol, net } from 'electron';
```

Add before `app.whenReady()`:
```ts
protocol.registerSchemesAsPrivileged([
  { scheme: 'launcher-file', privileges: { bypassCSP: true, stream: true } },
]);
```

Inside `app.whenReady().then(...)`, after `registerIpcHandlers(ipcMain)`:
```ts
protocol.handle('launcher-file', (request) => {
  const filePath = decodeURIComponent(request.url.replace('launcher-file://', ''));
  return net.fetch('file://' + filePath);
});
```

**Verify:** Protocol is registered before any window is created.

### Step 2 — Add worlds IPC handlers

**File:** `src/main/ipc-handlers.ts`

Add a new `WorldsStore` instance near the top alongside other `Store` instances
(or inline in the handlers — use inline since it's simple):

```ts
// In registerIpcHandlers, at the top of the function:
const worldsStore = new Store<{ notes: Record<string, string> }>({
  name: 'worlds',
  defaults: { notes: {} },
});
```

Then add handlers at the bottom of `registerIpcHandlers` (before the closing brace):

```ts
// ── Worlds ─────────────────────────────────────────────────────

ipcMain.handle('worlds:list', async (_e, instanceId: string) => {
  const instance = await instanceManager.get(instanceId);
  const savesDir = path.join(instance.gameDirectory, 'saves');

  try {
    const entries = await fs.readdir(savesDir, { withFileTypes: true });
    const worlds = await Promise.all(
      entries
        .filter((e) => e.isDirectory())
        .map(async (e) => {
          const stat = await fs.stat(path.join(savesDir, e.name));
          return { folderName: e.name, lastModified: stat.mtimeMs };
        }),
    );
    return worlds.sort((a, b) => b.lastModified - a.lastModified);
  } catch {
    return []; // saves dir doesn't exist yet (instance never launched)
  }
});

ipcMain.handle('worlds:get-notes', (_e, instanceId: string, folderName: string) => {
  const key = `notes.${instanceId}__${folderName}` as any;
  return (worldsStore.get(key) as string) ?? '';
});

ipcMain.handle('worlds:set-notes', (_e, instanceId: string, folderName: string, notes: string) => {
  const key = `notes.${instanceId}__${folderName}` as any;
  worldsStore.set(key, notes);
  return { success: true };
});

ipcMain.handle('worlds:list-screenshots', async (_e, instanceId: string) => {
  const instance = await instanceManager.get(instanceId);
  const screenshotsDir = path.join(instance.gameDirectory, 'screenshots');

  try {
    const entries = await fs.readdir(screenshotsDir);
    const pngs = entries.filter((f) => f.toLowerCase().endsWith('.png'));
    return pngs.map((fileName) => ({
      fileName,
      filePath: path.join(screenshotsDir, fileName),
    }));
  } catch {
    return [];
  }
});
```

Add `Store` import at top of file if not already present:
```ts
import Store from 'electron-store';
```
