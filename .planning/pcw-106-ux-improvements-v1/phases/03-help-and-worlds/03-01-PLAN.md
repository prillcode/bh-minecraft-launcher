# Plan: 03-01 — Help Panel (react-markdown + HelpPage + route + sidebar)

**Phase:** 03 — New Bottom Panel Group: Help + My Worlds
**Work:** pcw-106 ux-improvements-v1

## Context

Adds a Help panel with in-app launcher documentation, rendered from a Markdown string.
Also wires the new bottom sidebar group (Help + My Worlds nav items above Sign Out).
My Worlds nav item added here but its page is built in 03-03.

**Approach:**
- `react-markdown` + `remark-gfm` render Markdown in the renderer (pure JS, Electron-safe)
- Help content lives in `src/renderer/content/help.md`, imported as a raw string via Vite's `?raw` suffix
- Sidebar gets a second nav group below the Play section, above Sign Out
- Routes `/help` and `/worlds` added to App.tsx

No new IPC handlers needed for Help — all content is bundled.

## Dependencies to Add

```bash
pnpm add react-markdown remark-gfm
```

Both are pure ESM packages. `react-markdown` v9+ is ESM-only; Vite handles this fine in the renderer.

## Files to Create

- `src/renderer/content/help.md` — help content
- `src/renderer/components/help/HelpPage.tsx` — renders the markdown
- `src/renderer/components/worlds/WorldsPage.tsx` — placeholder (implemented in 03-03)

## Files to Modify

- [src/renderer/App.tsx](../../../../src/renderer/App.tsx) — add `/help` and `/worlds` routes

### Step 2 — Create help content

**File:** `src/renderer/content/help.md`

Write launcher documentation covering:
- **Getting Started** — what BlockHaven Launcher is, first launch
- **Instances** — creating, editing, deleting, mod loader selection
- **Mods** — browsing categories, searching Modrinth, installing, enabling/disabling
- **Shaders** — requirements (Iris + Sodium), installing from Modrinth or local file
- **Settings** — memory, Java path, close on launch, resolution
- **Troubleshooting** — game fails to launch, Java not found, mod conflicts

Keep it concise with `##` section headers and `###` subsections.

---

### Step 3 — Create `HelpPage` component

**File:** `src/renderer/components/help/HelpPage.tsx`

```tsx
import React from 'react';
import ReactMarkdown from 'react-markdown';
import remarkGfm from 'remark-gfm';
import helpContent from '../../content/help.md?raw';

export function HelpPage() {
  return (
    <div className="help-page">
      <div className="help-page__content">
        <ReactMarkdown remarkPlugins={[remarkGfm]}>
          {helpContent}
        </ReactMarkdown>
      </div>
    </div>
  );
}
```

### Step 6 — Update `Sidebar` with bottom nav group

**File:** `src/renderer/components/layout/Sidebar.tsx`

Add a second nav items constant for the bottom group:
```ts
const BOTTOM_NAV_ITEMS = [
  { to: '/worlds', label: 'My Worlds', icon: '🌍' },
  { to: '/help',   label: 'Help',      icon: '❓' },
];
```

In the JSX, add a new `<ul>` between `sidebar__play` and `sidebar__footer`:
```tsx
<ul className="sidebar__nav sidebar__nav--bottom">
  {BOTTOM_NAV_ITEMS.map((item) => (
    <li key={item.to}>
      <NavLink
        to={item.to}
        className={({ isActive }) =>
          `sidebar__link ${isActive ? 'sidebar__link--active' : ''}`
        }
      >
        <span className="sidebar__icon">{item.icon}</span>
        <span className="sidebar__label">{item.label}</span>
      </NavLink>
    </li>
  ))}
</ul>
```
