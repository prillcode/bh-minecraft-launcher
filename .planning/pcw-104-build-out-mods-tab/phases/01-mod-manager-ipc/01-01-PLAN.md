  <action>
Create `src/core/mods/mod-manager.ts`. Mirror the InstanceManager pattern.

Store configuration: `name: 'mods'`, `defaults: { mods: {} }`
Store type: `Store<{ mods: Record<string, Record<string, InstalledMod>> }>`
Keys: outer = instanceId, inner = projectId (InstalledMod.id)

**Methods:**

`list(instanceId: string): Promise<InstalledMod[]>`
  — return `Object.values(this.store.get('mods.' + instanceId as any) ?? {})` sorted by `installedAt` descending

`get(instanceId: string, projectId: string): Promise<InstalledMod | undefined>`
  — return `this.store.get('mods.' + instanceId + '.' + projectId as any) ?? undefined`

`add(instanceId: string, mod: InstalledMod): Promise<void>`
  — `this.store.set('mods.' + instanceId + '.' + mod.id as any, mod)`
  — log: `logger.info('Installed mod "' + mod.name + '" (' + mod.id + ') into instance ' + instanceId)`

`remove(instanceId: string, projectId: string): Promise<void>`
  — `this.store.delete('mods.' + instanceId + '.' + projectId as any)`
  — log the removal

`isInstalled(instanceId: string, projectId: string): Promise<boolean>`
  — return `!!(await this.get(instanceId, projectId))`

Imports: `import Store from 'electron-store'`, `import type { InstalledMod } from './types'`, `import { logger } from '../utils/logger'`.

Note: electron-store's nested key access via dot notation requires casting the path `as any` to satisfy TypeScript — this is the established pattern in InstanceManager (`this.store.delete('instances.' + id as any)`).
